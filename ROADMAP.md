# rustMH (ex-goMH) - Миграция на Rust

## Цели проекта
- Заменить консольный интерфейс (TUI) на полноценный touch-friendly GUI (`egui`).
- Оставить приложение портативным единым `.exe` файлом.
- Перевести системную логику Windows, настройку кассового ПО (iiko) и ККТ на Rust (`tokio` + WinAPI).
